# 🏯 修仙群侠传

> 一个由多 AI Agent 驱动的活修仙世界 · Android 原生文字游戏

---

## 🌟 这是什么？

**修仙群侠传**不是普通的文字游戏。它底层有多个 AI Agent（智能体），每个核心 NPC 都有自己的**性格、记忆、目标**，会自主做决策——今天去修炼还是去寻宝？跟谁结盟跟谁翻脸？

你不在的时候，江湖也在运转。

## ✨ 核心特色

| 特性 | 说明 |
|------|------|
| 🧠 **多Agent驱动** | 20+核心NPC由LLM驱动，有自己的个性和目标 |
| 🌍 **活的江湖** | 你离线时世界也在演化，上线后看到"江湖又变了" |
| 🏯 **20+宗门** | 每个宗门有自己的势力、外交、资源，关系动态变化 |
| 💰 **自配API** | 用自己的 API Key，费用透明，月均几毛到几块钱 |
| 📱 **纯本地存档** | 数据存在你手机里，可导出分享，隐私无忧 |

## 🚀 快速开始

### 前置条件

- Android Studio (最新版)
- 一个 LLM API Key（OpenAI / DeepSeek / 硅基流动等）

### 构建

```bash
git clone https://github.com/w1394847159/xianxia-qunxia.git
cd xianxia-qunxia
# 用 Android Studio 打开项目目录
# 点击 Run 即可
```

### 首次使用

1. 打开 App → 输入你的道号 → 踏入江湖
2. 去**设置** → **LLM API 配置** 填入你的 API Key 和接口地址
3. 回到江湖，你将看到 NPC 们开始行动了

## 🏗️ 项目架构

```
app/
├── game/              # 游戏引擎核心
│   ├── engine/        # 世界时间轴、事件系统、NPC决策器
│   ├── npc/           # NPC档案、行为树
│   ├── faction/       # 宗门系统、外交关系
│   └── world/         # 世界状态
├── agent/             # LLM Agent 层
│   ├── LlmClient.kt   # API调用封装
│   ├── TokenTracker.kt # Token统计
│   └── TokenEstimator.kt # 费用预估
├── data/              # 持久化层 (Room)
│   ├── db/            # 数据库
│   ├── model/         # 数据实体
│   └── repository/    # 仓库 + 初始世界数据
├── settings/          # 设置管理
└── ui/                # Jetpack Compose 界面
    └── screens/       # 所有页面
```

## 💡 费用参考

核心NPC使用LLM做决策会消耗Token。以下为预估（按 gpt-4o-mini）：

| 模式 | 月Token | 预估月费 |
|------|---------|---------|
| 悠闲 (10NPC×日2次) | ~90万 | ~$0.14 |
| 标准 (20NPC×日1次) | ~108万 | ~$0.18 |
| 热闹 (30NPC×日1次) | ~180万 | ~$0.27 |
| 风云 (40NPC×日1.5次) | ~396万 | ~$0.59 |

> 用 DeepSeek 或国产模型更便宜，甚至免费额度就够了。
